# C# Calculator App

Bu proje C# kullanılarak geliştirilmiş basit bir konsol hesap makinesi uygulamasıdır.

## Özellikler

* Toplama, çıkarma, çarpma, bölme işlemleri
* Kullanıcıdan veri alma
* Hatalı girişlere karşı kontrol (try-catch)
* İşleme devam edebilme

## Kullanılan Teknolojiler

* C#
* .NET Console

## Amaç

C# dilinde temel programlama mantığını geliştirmek ve pratik yapmak.
